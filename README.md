# EscrowStack – Stock Broker Client Dashboard

##  Project Overview

EscrowStack is a real-time Stock Broker Client Web Dashboard that allows users to log in, subscribe to selected stocks, and view continuously updating stock prices without refreshing the page. The application demonstrates real-time communication using WebSockets and a modern full-stack web architecture.

##  Tech Stack

**Frontend**

* React.js
* React Router
* HTML
* CSS

**Backend**

* Node.js
* Express.js
* Socket.IO (WebSockets)

**Deployment**

* Frontend: Render (Static Site)
* Backend: Render (Web Service)
